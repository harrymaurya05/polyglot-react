# @polyglot/react Examples

This directory contains example implementations of the @polyglot/react library.

## Basic Usage Example

See `basic-example/` for a simple implementation showing:

- Vite plugin configuration
- TranslateProvider setup
- useTranslate hook usage
- Language switching with useTranslator

## Run Examples

```bash
cd examples/basic-example
npm install
npm run dev
```

## What You'll Learn

1. **Text Extraction**: How the Vite plugin automatically extracts translatable text
2. **Translation Setup**: Configuring translation providers (Google, DeepL, AWS)
3. **Caching**: How translations are cached for offline use
4. **Dynamic Content**: Translating user-generated content with useTranslateDynamic
5. **Language Switching**: Implementing a language selector component

## Example Structure

```
basic-example/
├── src/
│   ├── App.tsx              # Main app with TranslateProvider
│   ├── components/
│   │   ├── Header.tsx       # Example using useTranslate
│   │   └── LanguageSwitcher.tsx  # Language selector
│   └── translations/
│       └── texts.json       # Auto-generated by plugin
├── vite.config.ts           # Vite plugin configuration
└── package.json
```
